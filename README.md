I'm a Software Engineering student at DTU who likes building things across the ML stack, from a rough idea to something that actually runs. My current internship work is in voice biometrics and audio deepfake detection, but I like to move around: agentic systems, RAG, computer vision, and the occasional full-stack side project.

- Comfortable across the ML lifecycle: research, training, evaluation, and deployment.
